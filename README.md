# Layout Inventory

居家空間 Layout 與物品庫存管理系統。

## 使用方式

直接開啟 `index.html` 即可使用。

如果放到 GitHub Pages，請把這個資料夾內的所有檔案放在 repository 根目錄，然後到 GitHub 的 Pages 設定中選擇從 `main` branch 的 root 發布。

## 主要檔案

- `index.html`：主頁面
- `app.js`：系統功能與資料邏輯
- `styles.css`：介面樣式
- `assets/`：家具、門窗、設備圖示
- `data/`：可放同步或備份資料

## 資料提醒

實際庫存資料可能包含個人物品、房間配置或備份內容。公開上傳 GitHub 前，建議不要把自己的 `layout-inventory-data.json` 放進 repository。

目前這份整理版已保留 `data/` 資料夾，但沒有放入個人 JSON 資料。
